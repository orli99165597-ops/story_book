# 📖 AI 이야기의 속삭임 — 엄마와 아들의 댄스 타임

> "함께 춤추는 행복한 거실의 기적"

엄마와 아들이 함께 훌라후프·댄스를 즐기다 웃음 가득한 방귀 사건까지 겪는 코믹 동화입니다.  
5~9세 어린이를 대상으로 한 **AI 인터랙티브 플립북** 웹 애플리케이션입니다.

## ✨ 주요 기능

### 📖 플립북 인터랙션

- **StPageFlip** 기반 실제 책처럼 넘기는 플립 애니메이션
- 왼쪽(이미지) + 오른쪽(텍스트) 2단 레이아웃 구성
- ◀▶ **좌우 화살표 버튼** — 화면 중앙 고정, 어린이도 쉽게 누를 수 있는 대형 버튼 (60x60px+)
- ⌨️ **키보드 지원** — ← → 화살표 키로 페이지 전환
- 📊 **진행률 표시바** — 현재 읽고 있는 위치를 그라데이션 바로 표시

### 🔊 TTS 음성 재생 & 동기화

- 🎧 **구연동화 오디오** — 전체 스토리 음성 재생 & 다운로드
- 📝 **문장별 하이라이트 동기화** — `sync.json` 타임스탬프 기반으로 현재 읽히는 문장 자동 강조
- 🔈 **배경음악(BGM)** 켜기/끄기 토글 버튼

### 🎨 사용자 맞춤 설정

- 🔤 **글자 크기 조절** — 아이 👦 / 엄마 👩 / 할머니 👵 3단계 폰트 사이즈 모드
- 🌙 **다크 모드** — 눈이 편한 어두운 테마 전환
- 🔖 **북마크 & 이어 읽기** — `localStorage` 기반 현재 페이지 저장 및 복원

### 🌟 비주얼 & 애니메이션

- ✨ **배경 별 파티클** — 몽환적인 우주 분위기 반짝이는 별 애니메이션
- 🌈 **그라데이션 글로우 타이틀** — 골드→핑크→퍼플 그라데이션의 빛나는 제목
- 📱 **반응형 레이아웃** — 모바일/태블릿/데스크톱 완벽 대응

### ♿ 접근성

- ARIA 레이블 적용 (`aria-label` 속성)
- 충분한 터치 영역 확보 (60x60px 이상)
- 시맨틱 HTML5 구조

## 🚀 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 바로 실행됩니다.

```bash
# 방법 1: 브라우저에서 직접 열기
open index.html        # macOS
start index.html       # Windows

# 방법 2: 로컬 서버 사용 (CORS 이슈 방지)
npx serve .
# 또는
python -m http.server 8080
```

> **참고**: 오디오 파일 재생을 위해 로컬 서버 사용을 권장합니다.

## 🛠️ 기술 상세

| 기술 요소                 | 설명                                                      |
| ------------------------- | --------------------------------------------------------- |
| **StPageFlip**            | CDN 연동, 양쪽 두 페이지 펼침 모드 (`usePortrait: false`) |
| **CSS Custom Properties** | `--fs-body`, `--fs-title`, `--lh-body` 디자인 토큰        |
| **TTS 동기화**            | `sync.json` 타임스탬프 + `timeupdate` 이벤트 기반         |
| **반응형 책 크기**        | `getBookSize()` 함수로 뷰포트별 자동 조정                 |
| **localStorage**          | 북마크(이어 읽기), 글자 크기, 다크모드 설정 저장          |
| **CSS 애니메이션**        | `@keyframes` 기반 별 반짝임, 제목 글로우, 바운스 효과     |
