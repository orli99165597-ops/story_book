# 📖 AI 이야기의 속삭임 — 방귀와 다이어트

> "함께 춤추는 행복한 거실의 기적"

엄마와 아들이 함께 훌라후프·댄스를 즐기다 웃음 가득한 방귀 사건까지 겪는 코믹 동화입니다.  
5~9세 어린이를 대상으로 한 **AI 인터랙티브 플립북** 웹 애플리케이션입니다.

---

## 🎯 프로젝트 개요

| 항목                | 내용                                               |
| ------------------- | -------------------------------------------------- |
| **장르**            | 어린이 코믹 동화                                   |
| **대상 연령**       | 5 ~ 9세                                            |
| **총 페이지**       | 표지 + 8페이지 + 뒤표지 (총 10면)                  |
| **기술 스택**       | HTML5, CSS3, Vanilla JavaScript (ES6+)             |
| **핵심 라이브러리** | [StPageFlip](https://github.com/Nodlik/StPageFlip) |

## ✨ 주요 기능

- 📖 **StPageFlip** 기반 실제 책처럼 넘기는 플립북 애니메이션
- ◀▶ **좌우 화살표 버튼** — 화면 중앙 고정, 어린이도 쉽게 누를 수 있는 대형 버튼
- ⌨️ **키보드 지원** — ← → 화살표 키로 페이지 전환
- 🎧 **오디오 플레이어** — 구연동화 듣기 & 다운로드
- 📊 **진행률 표시바** — 현재 읽고 있는 위치 표시
- 📱 **반응형 레이아웃** — 모바일/태블릿/데스크톱 대응
- 🌟 **배경 별 파티클 애니메이션** — 몽환적인 우주 분위기
- ♿ **접근성** — ARIA 레이블 적용

## 🛠️ 기술 상세

- **StPageFlip** CDN 연동 — 양쪽 두 페이지 펼침 모드 (`usePortrait: false`)
- **CSS Custom Properties** 기반 디자인 토큰
- **Intersection Observer** 없이 순수 JS 이벤트 기반 페이지 감지
- **반응형 책 크기** — `getBookSize()` 함수로 뷰포트별 자동 조정
